## DB : oracle 21

### 1. `blogmember`
블로그 회원 정보를 저장하는 테이블

| 필드명       | 데이터 유형                       | 제약 조건                           |
|--------------|----------------------------------|-------------------------------------|
| `userId`     | `NUMBER`                         | `GENERATED ALWAYS AS IDENTITY`      |
| `username`   | `VARCHAR(100)`                   | `NOT NULL`                          |
| `useremail`  | `VARCHAR(100)`                   | `NOT NULL`                          |
| `userpassword`| `VARCHAR(200)`                  | `NOT NULL`                          |
| `profiletext`| `VARCHAR(255)`                   | `DEFAULT ''`                        |
| `dateTime`   | `TIMESTAMP`                      | `DEFAULT CURRENT_TIMESTAMP`         |
| **Primary Key** | `userId, useremail`          |                                     |

### 2. `blogboard`
블로그 게시글 정보를 저장하는 테이블

| 필드명       | 데이터 유형                       | 제약 조건                           |
|--------------|----------------------------------|-------------------------------------|
| `blogId`     | `NUMBER`                         | `GENERATED BY DEFAULT AS IDENTITY`  |
| `writer`     | `VARCHAR2(100)`                  | `NOT NULL`                          |
| `title`      | `VARCHAR(255)`                   | `NOT NULL`                          |
| `blogContent`| `CLOB`                           | `NOT NULL`                          |
| `blogviews`  | `NUMBER`                         |                                     |
| `dateTime`   | `TIMESTAMP`                      | `DEFAULT CURRENT_TIMESTAMP`         |
| **Primary Key** | `blogId`                      |                                     |

### 3. `blogcomment`
블로그 게시글에 대한 댓글 정보를 저장하는 테이블

| 필드명           | 데이터 유형                       | 제약 조건                                   |
|------------------|----------------------------------|---------------------------------------------|
| `commentId`      | `NUMBER`                         | `GENERATED BY DEFAULT AS IDENTITY`          |
| `blogId`         | `NUMBER`                         | `NOT NULL`                                  |
| `userId`         | `NUMBER`                         | `NOT NULL`                                  |
| `useremail`      | `VARCHAR(100)`                   | `NOT NULL`                                  |
| `commentContent` | `CLOB`                           | `NOT NULL`                                  |
| `dateTime`       | `TIMESTAMP`                      | `DEFAULT CURRENT_TIMESTAMP`                 |
| **Foreign Keys**  | `blogId` REFERENCES blogboard(blogId) ON DELETE CASCADE  |
|                  | `userId, useremail` REFERENCES blogmember(userId, useremail) ON DELETE CASCADE |

### 4. `bloglike`
블로그 게시글에 대한 좋아요 정보를 저장하는 테이블

| 필드명       | 데이터 유형                       | 제약 조건                                   |
|--------------|----------------------------------|---------------------------------------------|
| `likeId`     | `NUMBER`                         | `GENERATED BY DEFAULT AS IDENTITY`          |
| `blogId`     | `NUMBER`                         | `NOT NULL`                                  |
| `userId`     | `NUMBER`                         | `NOT NULL`                                  |
| `useremail`  | `VARCHAR(100)`                   | `NOT NULL`                                  |
| `dateTime`   | `TIMESTAMP`                      | `DEFAULT CURRENT_TIMESTAMP`                 |
| **Foreign Keys**  | `blogId` REFERENCES blogboard(blogId) ON DELETE CASCADE  |
|                  | `userId, useremail` REFERENCES blogmember(userId, useremail) ON DELETE CASCADE |
| **Unique Key** | `(blogId, userId, useremail)`  |                                             |

### 5. `blognotice`
블로그 공지사항 정보를 저장하는 테이블

| 필드명           | 데이터 유형                       | 제약 조건                                   |
|------------------|----------------------------------|---------------------------------------------|
| `noticeid`       | `NUMBER`                         | `GENERATED BY DEFAULT AS IDENTITY`          |
| `noticetitle`    | `VARCHAR2(1500)`                 | `NOT NULL`                                  |
| `noticecontent`  | `CLOB`                           | `NOT NULL`                                  |
| `dateTime`       | `TIMESTAMP`                      | `DEFAULT CURRENT_TIMESTAMP`                 |
